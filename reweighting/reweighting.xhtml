Prediction Definitions

  • Batch Linear Prediction `y[t] ~ \hat{y}[t] = \sum_{i = 1}^pc_k x[t - k]`
  • Sequential Linear Prediction `y[t] ~ \hat{y}[t] = \sum_{i = 1}^pc_k^{t-1} x[t - k]`
  • Square Loss `l_n\left(y, \hat{y}\right) = \sum_{i = 1}^n (y[i] - \hat{y}[i] )^2`
  • Experts `\hat{y}_1, \ldots, \hat{y}_m`

Reweighing

  • `\hat{y}_u[t] = \sum_{k = 1}^m u_k[t] \hat{y}_k[t]`

Expert's "Probability"

  • An expert's probability is `P_k(y^n) = B \exp(-\frac{1}{2c}l_n(y, \hat{y}_k))`
    • "Unlikely" sequences are high-loss ones
  • Analogous conditional probability `P_k(y[n]|y^n) = \frac{}{} = B \exp(-\frac{1}{2c}l_n(y, \hat{y}_k))`
